Steps to Run the program

1. There is a two java file named 'MinConflict.java' in folder attched in canvas which is a driver class and calls randomRestart Class to implements the Hill Climbing  with random restart to solve n-queen problem

2. Run it using any Java IDE.

3. Program prints the total execution time No. of restarts and No of states generated in console.
